Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used for set HTML code in new created Tag or set an HTML code in existing Tag. innerText is used for display visible text in the html tag or set a new text value inside the tag. on the other hend innerContent works somewhat like innerText, but there are some differences. innerText returns only visible text of html element on the other hend textContent returns not only visible value but also hidden value. and it's also use for set a new text into the HTML tag..

Question-4: What is event bubbling in the DOM?

Bubbling is a propagation system that moves from the HTML root parent to the selected element, passing through each step one by one.
